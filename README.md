#### 项目介绍

未来软件工作室招新报名面试系统，用于工作室招新使用，主要功能有招新报名、上传笔试图片、查看面试题库、现场面试打分、查看打分结果，名单批量导出、题库管理等，极大方便了工作室招新面试相关事宜。
